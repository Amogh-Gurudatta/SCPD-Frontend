A high-performance, neon-noir dual-theme spatial planning dashboard inspired by the grit and vibrant lights of Las Vegas. SCPD v2 leverages modern web technologies to provide a seamless geospatial experience with real-time data visualization.

---

## 🚀 Built With

- **Framework**: [Next.js 16 (App Router)](https://nextjs.org/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Mapping**: [Leaflet](https://leafletjs.com/) & [React Leaflet](https://react-leaflet.js.org/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Utilities**: `clsx`, `tailwind-merge`

## 🛠️ Getting Started

### Prerequisites

- Node.js 18.x or higher
- npm or yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/scpdv2.git
   cd scpdv2
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open the application:**
   Navigate to [http://localhost:3000](http://localhost:3000) to see the results.

## 🎨 Design System

- **Dual-Theme**: Optimized for both Light and Dark modes.
- **Geospatial Focus**: Integrated Leaflet patterns for complex map interactions.
- **Glassmorphism**: Modern UI components with subtle transparency and blur effects.
